# GitHub Portfolio

This is a Next.js portfolio demo for Hari Prasad Nalajala.

## Quick Start

1. **Install Node.js & npm** if not already installed.
2. Install dependencies:
   ```${'
'}npm install
   ```
3. Install required peer dependencies:
   ```${'
'}npm install framer-motion lucide-react
   ```
4. If not already present, set up Tailwind CSS ([setup guide](https://tailwindcss.com/docs/guides/nextjs)).
5. Start the dev server:
   ```${'
'}npm run dev
   ```
6. Open [http://localhost:3000](http://localhost:3000).

---

## One-Click Deploy: Vercel

- Go to https://vercel.com/import
- Sign in with your GitHub
- Import this `github-portfolio` repo
- All deploy settings will be autoconfigured for Next.js
- You’ll get your personal portfolio preview link after the build!
